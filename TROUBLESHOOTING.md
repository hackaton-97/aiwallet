# Устранение проблем

## Ошибка "Not Found" при открытии signup.html

### Если используете Express сервер:

1. Убедитесь, что сервер запущен:
   ```bash
   node server.js
   ```

2. Откройте в браузере: `http://localhost:3000/signup.html`

3. Проверьте консоль сервера на наличие ошибок

### Если используете Live Server:

1. Убедитесь, что открываете файл через Live Server (не напрямую через файловую систему)

2. Правой кнопкой на `signup.html` → "Open with Live Server"

3. URL должен быть: `http://localhost:5500/signup.html` (или другой порт)

### Если используете GitHub Pages:

1. Убедитесь, что файл `.nojekyll` присутствует в корне репозитория

2. Проверьте, что все файлы загружены в репозиторий

3. URL должен быть: `https://[username].github.io/[repo-name]/signup.html`

### Общие проверки:

1. Убедитесь, что файл `signup.html` существует в корне проекта

2. Проверьте, что файлы `api-wrapper.js` и `script.js` находятся в той же папке

3. Откройте консоль браузера (F12) и проверьте наличие ошибок

4. Проверьте Network tab в DevTools - возможно, какой-то файл не загружается

### Если проблема сохраняется:

1. Очистите кэш браузера (Ctrl+Shift+Delete)

2. Попробуйте открыть в режиме инкогнито

3. Проверьте, что все пути к файлам относительные (не начинаются с `/`)

4. Убедитесь, что нет конфликтов с другими серверами на портах 3000, 5500 и т.д.
